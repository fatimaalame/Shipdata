##Description générale du projet

Le projet ShipData porte sur le domaine du transport maritime. 
L’idée est de créer une base de données relationnelle capable de représenter, de manière simplifiée mais réaliste, les informations principales liées aux voyages de navires entre différents ports. 
Le transport maritime est un domaine concret, utilisé quotidiennement pour déplacer des marchandises à grande échelle. 
Notre base de données ne cherche donc pas à couvrir toute la complexité du secteur, mais à modéliser ses éléments essentiels de façon claire et exploitable.

La base stocke des informations sur les compagnies maritimes, les navires, les ports, les routes maritimes, les voyages et les cargaisons transportées. 
Par exemple, elle permet de savoir à quelle compagnie appartient un navire, entre quels ports un voyage est effectué, quel type de marchandise est transporté, ou encore quel est le statut d’un voyage : prévu, en cours, terminé ou annulé.

L’objectif principal de ShipData est de faciliter la consultation et l’analyse de ces données. 
Un utilisateur peut rechercher les voyages liés à un port, identifier les navires d’une compagnie, consulter les cargaisons transportées ou comparer l’activité des différentes routes. 
La base peut donc être utile pour un employé logistique, un employé portuaire, un responsable de compagnie maritime ou un administrateur chargé de maintenir les données à jour.

Ainsi, ShipData propose une représentation structurée d’un domaine réel, tout en restant assez simple pour être comprise, interrogée et maintenue efficacement.
